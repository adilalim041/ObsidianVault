# https://github.com/rbren/rss-parser

Score: 8.1/10

News.AI's content factory needs reliable RSS feed parsing for content aggregation, and this library's dual callback/Promise API with HTTP caching (ETag/If-Modified-Since support) directly addresses the 'API reliability' backlog item. The field mapping system in lib/fields.js provides exactly the data normalization needed for multi-source content ingestion.
