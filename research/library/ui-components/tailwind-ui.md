# Tailwind UI / Tailwind Plus

## What it is

Premium (paid) collection of professionally designed Tailwind CSS components and templates from the Tailwind team itself. Hundreds of UI blocks (hero sections, pricing tables, dashboards, marketing pages) and full templates.

## License

**Paid commercial license.** ~$300 one-time for personal license. Once bought, you can use the components in unlimited personal/client projects.

## Used for

Future consideration. Not needed now — shadcn/ui + Tremor + Lucide cover 90% of dashboard needs for free.

## When it would be worth buying

- When launching a real public-facing landing page for News.AI brand or Omoikiri SaaS template
- When the templates would save a few days of design work that's worth more than $300
- When making a client pitch and you want it to look investor-grade

## Score: 6/10 for Adil

Not now. Worth it later when there's a real landing page or template product to ship.

## Alternatives (free)

- **shadcn/ui examples** — has dashboard templates (free)
- **Tremor's free dashboards** — see their docs
- **Flowbite** — Tailwind components, has free tier
