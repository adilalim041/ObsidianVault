# OpenGameArt.org

## What it is

A community-driven aggregator of free game assets — sprites, tile sets, music, sound effects, models. Uploaded by many different artists.

## License

**Varies per asset** — must check each one individually. Common licenses:
- **CC0** — totally free
- **CC-BY** — free, attribution required
- **CC-BY-SA** — free, attribution + your work must be ShareAlike
- **GPL** — careful, see below

## Used for

When Kenney doesn't have what you need.

## ⚠️ Critical warning

**Always read the license tab on each asset page before using.** Some popular assets are under CC-BY-SA or even GPL — using them requires you to release your own work under the same license. This is incompatible with most commercial products.

**Default rule for Adil: only use CC0 assets from this site.** Filter by license before downloading.

## How to use

1. Go to https://opengameart.org
2. Use the search filters and **set License = CC0** to be safe
3. Read the license tab on the asset page
4. Download and credit if required
