# Kenney.nl — game and UI assets

## What it is

A massive collection of free game assets created by Kenney Vleugels: sprites, tile sets, UI packs, characters, fonts, sound effects, 3D models. All released into the **public domain**.

## License

**CC0 (Creative Commons Zero / Public Domain).** No restrictions whatsoever. No attribution required. You can use, modify, redistribute, sell — anything.

## Used for

- **Nexus.AI** — if Adil ever does a "Jarvis-style" demo with on-screen visuals, Kenney has UI panel sprites
- **Future game/visual experiments** — when Adil wanted to ask "can I use sprites from games" — yes, here, legally and freely
- **Placeholder graphics** during prototyping when you want something nicer than circles

## How to use

1. Go to https://kenney.nl/assets
2. Browse by category (Game UI, 2D, 3D, Audio, Tiles, etc.)
3. Download the pack (ZIP)
4. Drop files into your project

## Score: 9/10 for the niche it covers

The single biggest source of legally-free game art on the internet. If you ever experiment with games, mini-games, or visual demos — start here, never copy from a copyrighted game.

## Why this matters for Adil's question

Adil specifically asked: "if I want to make a game, can I use sprites and characters from games on GitHub?" Answer: **almost never legally** — most game sprites uploaded to GitHub are copyrighted and the upload itself is infringement. **Kenney's collection is the legal answer** to that question.

## Alternatives

- **OpenGameArt.org** — aggregator, per-asset licenses vary, **always check each one**
- **itch.io free assets** — many CC0 packs, also varies per asset
- **Game-icons.net** — 4000+ free icons specifically for games (CC-BY)
