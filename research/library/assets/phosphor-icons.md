# Phosphor Icons

## What it is

A flexible icon family with 1200+ icons in 6 different weights (Thin, Light, Regular, Bold, Fill, Duotone). Same icon, different stylistic intensity.

## License

**MIT.**

## Used for

Alternative to Lucide when you specifically want the duotone or bold visual style for emphasis.

## How to use

```bash
npm i @phosphor-icons/react
```

```tsx
import { Heart, Star, Lightning } from '@phosphor-icons/react'

<Heart size={24} weight="fill" color="red" />
<Star size={24} weight="duotone" />
```

Use Lucide as default; reach for Phosphor when you want more visual variation in a single screen.
