# Tabler Icons

## What it is

A massive icon set with 4000+ icons. Largest free icon set in the React ecosystem.

## License

**MIT.**

## Used for

When Lucide doesn't have the specific icon you need (rare but happens), Tabler probably does.

## How to use

```bash
npm i @tabler/icons-react
```

```tsx
import { IconBrandWhatsapp, IconRobot, IconChartBar } from '@tabler/icons-react'

<IconBrandWhatsapp size={24} />
```

## Score: 8/10 for Adil

The brand icons (WhatsApp, Telegram, Instagram, TikTok, Meta) are particularly useful for Adil's projects — Lucide has fewer brand icons.
